Welcome to Grey Prediction System Online Computing Platform

欢迎来到灰色预测在线计算平台，你可以在本平台选择一个模型去拟合你现有的数据并做出预测。现在本平台只实现了基本的GM(1,1)模型和一些基于数值积分算法优化的GM(1,1)模型。我们将上线更多典型的灰色预测模型。敬请期待！

This is a online computing platform of Grey Prediction System. You can choose a Model to simulate your own data and make prediction with the model you chosen. Now this online platform only implemented the basic GM(1,1) model and some optimized GM(1,1) models with the numerical integration algorithms. We'll implement more typical Grey System model in the future!
